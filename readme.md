# Confetti Overlay

This project is a lightweight OBS browser overlay for animated confetti effects. Each HTML file can load a different confetti style.

### [Download it here](https://github.com/DazXL/Confetti/releases)

## Setup

1. Download the project files.
2. Extract all files into a single folder.
3. In OBS, add a new Browser Source.
4. Enable Local File.
5. Browse to the folder and choose one of the HTML files.
6. Check both "Shut down source when not visible" and "Refresh browser when scene become active". (you may have to scroll down a bit)

## Notes

- Each HTML file can be configured to load a different confetti effect.
- The overlay is intended to stay lightweight and simple to reuse in OBS scenes.
