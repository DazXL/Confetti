Confetti Overlay

This project is a lightweight OBS browser overlay for animated confetti effects. Each HTML file loads a different effect, so you can switch styles by selecting a different page.

Setup

  1. Download the project files.
  2. Extract all files into a single folder.
  3. In OBS, add a new Browser Source.
  4. Enable Local File.
  5. Choose one of the sample HTML files.

Credits
